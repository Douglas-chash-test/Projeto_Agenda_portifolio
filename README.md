# Projeto_Agenda_portifolio
Este projeto é uma aplicação de gerenciamento de tarefas (To-Do List) desenvolvida para consolidar conhecimentos em operações CRUD, autenticação de usuários e persistência de dados

# Funcionalidades
Autenticação Completa: Sistema de Login e Cadastro de novos usuários.
Segurança: Fluxo de redefinição de senha.
Gerenciamento de Tarefas (CRUD): Criação, leitura, edição e exclusão de tarefas.
Interface Intuitiva: Visualização das tarefas em formato de cards para melhor organização visual.
